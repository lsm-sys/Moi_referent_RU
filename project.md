Приложение получает на входе адрес франкоязычной статьи, выполняет парсинг и, по соответствующей кнопке, подключает AI и выполняет нужное действие.
Интерфейс:
Поле ввода URL франкоязычной статьи.
3 кнопки: «О чем статья?», «Пост для Дзен», «Пост для Telegram».
Блок для отображения результата, в котором происходит генерация нужного вида ответа.
